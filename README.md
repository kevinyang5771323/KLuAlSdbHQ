# Python学生信息管理系统 python1292

## 项目概述

本项目是一个基于Flask框架的学生信息管理系统，采用MySQL数据库进行数据存储，使用HTML、CSS和JavaScript进行前端页面设计。

## 技术栈

- 后端框架：Flask
- 数据库：MySQL
- 前端技术：HTML、CSS、JavaScript
- 开发工具：PyCharm、Navicat

## 功能模块

系统主要包含以下功能模块：

1. 学生信息管理
2. 学生成绩管理
3. 用户权限管理

## 系统角色

- 管理员：负责系统维护、用户管理和数据备份等操作
- 教师：负责录入和管理学生成绩
- 学生：查看个人信息和成绩

## 运行环境

- Python版本：3.x
- 操作系统：Windows、Linux或macOS
- 数据库：MySQL 5.x或更高版本

## 部署步骤

1. 安装Python环境和依赖包
2. 安装MySQL数据库
3. 导入数据库文件
4. 修改配置文件中的数据库连接信息
5. 运行Flask应用程序
